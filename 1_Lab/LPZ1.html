1. Структура веб-сайта

Спроектирована многостраничная структура сайта с главной страницей и разделами:

  • index.html - главная страница (данная страница)
  • about.html - информация об авторе
  • gallery.html - галерея изображений
  • contacts.html - контактная информация
  • references.html - полезные ресурсы

Навигация реализована через горизонтальное меню в шапке сайта. Все страницы имеют единый стиль оформления и структуру.

2. Основы языка HTML

HTML (HyperText Markup Language) — язык разметки гипертекста, используемый для создания документов, независимых от аппаратно-программной платформы. HTML — это не язык программирования, а описательный язык.

Важные особенности HTML:

  • Теги не чувствительны к регистру (но рекомендуется использовать нижний регистр)
  • Большинство тегов являются парными (имеют открывающий и закрывающий тег)
  • Теги могут быть вложенными, важно соблюдать порядок вложенности
  • Значения атрибутов рекомендуется заключать в кавычки

Пример структуры HTML-документа:

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Заголовок страницы</title>
</head>
<body>
  <h1>Основной заголовок</h1>
  <p>Текст абзаца</p>
</body>
</html>

4. Таблицы

Таблицы в HTML создаются с помощью тега <table>. Строки формируются тегом <tr>, ячейки заголовков — <th>, ячейки данных — <td>.

Пример таблицы: Сравнение версий HTML

Версия Год выпуска Ключевые особенности Поддержка медиа
HTML 4.01 1999 Стандартизация, разделение структуры и оформления Ограниченная
XHTML 1.0 2000 Строгий синтаксис на основе XML Ограниченная
HTML5 2014 Семантические теги, нативная поддержка мультимедиа Полная (видео, аудио, canvas)

Примечание: В современной веб-верстке таблицы рекомендуется использовать только для табличных данных. Для создания макетов страниц следует применять CSS-сетки и флексбоксы.

5. Фреймы (встраиваемые окна)

В современном HTML традиционные фреймы (<frameset>) считаются устаревшими. Вместо них рекомендуется использовать тег <iframe> для встраивания внешнего контента.

Тег <iframe> имеет следующие важные атрибуты:

  • src - адрес встраиваемого ресурса
  • width и height - размеры фрейма
  • title - описание содержимого для доступности
  • sandbox - ограничение функциональности для безопасности

Важно: При использовании iframe следует учитывать политику безопасности сайтов. Многие ресурсы запрещают встраивание через заголовок X-Frame-Options. Для учебных целей в данном примере используется карта с разрешением на встраивание.
